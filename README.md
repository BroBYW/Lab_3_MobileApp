## üöÄ Getting Started

### Prerequisites
- Visual Studio 2022 with .NET MAUI workload
- .NET 9.0 SDK
- Platform-specific SDKs (Android SDK, Xcode for iOS/macOS)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd lab2
```

2. Restore NuGet packages:
```bash
dotnet restore
```

3. Build the project:
```bash
dotnet build
```

4. Run the application:
```bash
dotnet run
```
